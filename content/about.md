I'm Viktor Maruna, a Data Engineer and Solution Architect (Data & Analytics) at [Adastra](https://adastracorp.com/) with over 10 years of experience specializing in enterprise data platforms, cloud architecture and analytics solutions.

In this blog, I share technical insights on data engineering, cloud architecture, and real-world solutions, along with lessons learned from architecting data platforms.

## Expertise

A decade of hands-on experience across the full data and cloud stack, with a focus on Microsoft Azure.

- **Data Analytics Platforms:** Microsoft Fabric, Azure Databricks, Azure Synapse Analytics, Microsoft Foundry
- **Cloud Platforms:** Azure (IaaS/PaaS, security, networking, governance), AWS
- **Cloud Architecture:** Solution design, Infrastructure as Code (Terraform), CI/CD, Azure DevOps
- **Data Architecture:** Data lakes, lakehouses, data warehouses, performance tuning and optimisation
- **Data Integration & Engineering:** Azure Data Factory, Azure Event Hubs, Kafka, Sqoop, Apache Spark
- **Databases & Storage:** Microsoft SQL Server, Azure SQL Database, Azure Cosmos DB, Azure Data Lake Storage, PostgreSQL
- **Programming & Scripting:** SQL, T-SQL, Python, Apache Spark, Bash, PowerShell
- **Big Data:** Hadoop (Cloudera, Hortonworks), Impala, Hive, HDFS
- **DevOps & Version Control:** Azure DevOps, GitHub, Git, GitHub Copilot
